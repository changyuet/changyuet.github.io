## 最近无意之中发现的 月亮播放器 Linux
在 LinuxMint/Ubuntu 系列系统中，使用如下 PPA 安装月亮播放器：
```
sudo add-apt-repository ppa:cos-lyk/moonsoft
sudo apt update
sudo apt install moonplayer
```
